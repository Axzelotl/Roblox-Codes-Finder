# Roblox Codes Finder
A tool used to quickly find and store redeemable codes in most Roblox games.

# About
A simple program which quickly searches for redeemable reward codes from a given Roblox game on beebom.com. Found codes are stored in a text document inside "Saved Codes" folder. Only new codes will be shown upon re-entering the game name and stored codes will not be shown again.
