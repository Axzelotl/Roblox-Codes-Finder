# 🔍 Roblox Code Finder
A tool used to quickly find and store redeemable codes in most Roblox games.

# 📄 About
A simple program which quickly searches for redeemable reward codes from a given Roblox game on [Beebom](https://beebom.com). Found codes are stored in a text document inside "Saved Codes" folder. Only new codes will be shown upon re-entering the game name and stored codes will not be shown again. 

# 📁 Installation 
In order to **install** the program:
- Download the [lastest release](https://github.com/Axzelotl/Roblox-Code-Finder/releases/tag/v1.0.0) of **Roblox Code Finder**.
- Extract the .zip folder.
  
And the program is ready to use!

# 🛠️ Usage
In order to **use** the program:
- Run **code_finder.exe**.
- Enter the name of the Roblox game you wish to find codes for.

And any **new** codes for the given game will be listed and saved in the "**Saved Codes**" folder.
